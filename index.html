Materias de 3er semestre



Submodulos del semestre

  1. Base De Datos

    2. Objetivos

    • 1.Introduccion a Access: El alumno se familiariza con la interfaz y herramientas de Microsoft Office.
    • 2.Formularios de Access: El estudiante generá una base de datos a la cual le generará los formularios pertinentes para un manejo optimo de los datos.
    • 3.Consuta de Access: El alumno generará las consultas pertinentes de una base de datos, con la finalidad de poder accesar de una manera mas facil a la información.

  2. Programacion I

    3. Objetivos

    • 1.¿Qué es un algoritmo?: Este submodulo se centrará en que el alumno tenga conociento de que es un algoritmo, asi como su creación a partir de un problema dado.
    • 2.Identificacion de un problema: Se enseñara al alumno a identificar la raiz del problema planteado, asi como a ser analítico y critico a la hora de tomar decisiones.
    • 3.Solución de un problema: El alumno aprende finalmente a dar la mejor solución posible al problema planteado, teniendo en cuentaque la solución debe de ser rápida, facil y sencilla.

  3. Hoja de Cálculo I

    4. Obejtivos

    • 1.Introducción a Excel: El alumno aprende en este submodulo a utilizar la interfaz y herramientas que ofrece Microsoft Excel.
    • 2.Operaciones en Excel: En este submodulo el estudiante vera como utlizar algunas operaciones básicasen una hoja de cálculo.
    • 3.Gráficas en Excel: El estudiante logrará identificar el uso de las gráficas de Microsoft Excel brinda en diferentes contextos.
